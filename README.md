# 📚 سیستم مدیریت کتابخانه شخصی (C++)

یک برنامه کنسولی به زبان C++17 برای مدیریت مجموعه کتاب‌های شخصی.
این پروژه برای درس برنامه‌سازی پیشرفته توسعه داده شده است.

## ✨ امکانات

- ➕ افزودن کتاب جدید
- ✏️ ویرایش کتاب با ISBN
- 🗑️ حذف کتاب با ISBN
- 🔍 جستجوی کتاب بر اساس عنوان، نویسنده یا ISBN
- 🔃 مرتب‌سازی کتاب‌ها بر اساس هر یک از فیلدها
- 📋 نمایش لیست کامل کتاب‌ها
- 💾   :**سه فرمت**ذخیره‌سازی در  
 - **Binary (.dat)** - (Native)فشرده، سریع، فرمت   
  - **CSV (.csv)** -  قابل باز شدن در اکسل، قابلیت خواندن بهتر  
  - **JSON (.json)** - استاندارد وب/API، ساختاریافته
- 📂 بارگذاری از فایل باینری و... در شروع برنامه
- ✅ اعتبارسنجی ورودی‌ها (ISBN، سال، فیلدهای خالی)
- 🛡️ مدیریت خطا با استثناهای سفارشی

## 📁 ساختار پروژه

```
personal-library/
```

## 📖 نحوه استفاده

1. در اولین اجرا، برنامه به طور خودکار یک فایل داده خالی می‌سازد.
2. از منوی اصلی، گزینه ۱ را برای افزودن کتاب انتخاب کنید.
3. ISBN باید ۱۰ یا ۱۳ رقم باشد (می‌توانید از خط تیره استفاده کنید).
4. سال انتشار باید بین ۱۴۵۰ و ۲۰۲۶ باشد.
5. قبل از خروج برای ذخیره داده ها از شما سوال پرسیده میشود .


## فرمت‌های فایل

### فرمت باینری (.dat)

رشته‌های با طول مشخص برای سازگاری بین پلتفرم‌ها:

```
[count: uint32_t]
برای هر کتاب:
  [len: uint32_t][title: chars]
  [len: uint32_t][author: chars]
  [len: uint32_t][isbn: chars]
  [year: int32_t]
```

### فرمت CSV (.csv)

مطابق با استاندارد RFC 4180 با escaping صحیح:

```csv
Title,Author,ISBN,Year
The C Programming Language,Brian Kernighan,9780131103627,1988
"Hello, World","Author ""Test""",1234567890,2020
```

### فرمت JSON (.json)

فرمت استاندارد JSON با escaping صحیح:

```json
{
  "books": [
    {
      "title": "The C Programming Language",
      "author": "Brian Kernighan",
      "isbn": "9780131103627",
      "year": 1988
    }
  ]
}
```

## 🏗️ معماری

### کلاس Book
نمایش یک کتاب با چهار ویژگی: title, author, ISBN, publicationYear.
دارای Getter/Setter با اعتبارسنجی و Operator Overloading.

### کلاس Library
مدیریت مجموعه‌ای از کتاب‌ها با استفاده از std::vector.
شامل عملیات CRUD، جستجو، مرتب‌سازی و مدیریت فایل.

### استثناهای سفارشی
برای مدیریت خطاهای مختلف (ISBN نامعتبر، کتاب تکراری، فایل و غیره).
